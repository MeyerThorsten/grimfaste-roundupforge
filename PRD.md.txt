# Amazon Roundup Finder

Build a web app that accepts up to 100 newline-separated "best of" keywords and returns up to 15 Amazon product candidates per keyword.

## Core workflow
1. User pastes up to 100 roundup keywords.
2. App builds Amazon search URLs from each keyword.
3. App uses ScrapeOwl to fetch Amazon search results pages.
4. App extracts candidate product links from search results.
5. App visits product pages and extracts structured product data using a reusable scrape profile.
6. App returns grouped results by keyword.
7. User can export results as JSON and CSV.

## Example input
best robotic pool cleaners for inground pools
best robotic pool cleaners for above ground pools
best cordless robotic pool cleaners

## Example output per keyword
- keyword
- searchUrl
- products[1..15]
  - title
  - asin if detectable
  - productUrl
  - affiliateUrl
  - imageUrl
  - extracted sections:
    - featureBullets
    - productDescription
    - productFacts
    - techDetails
    - reviews
  - mergedText
  - scrapeDebug info

## Scrape profile support
A scrape profile must support:
- domain
- title selector
- image selector
- comma-separated text selectors
- affiliate code
- enabled toggle

Default Amazon profile:
- domain: amazon.com
- title: #productTitle
- image: #imgTagWrapperId
- text selectors:
  #bookDescription,#productFactsDesktopExpander,#feature-bullets,#productDescription_feature_div,#prodDetails,#tech,.review-text,#productFactsDesktopExpander

## MVP features
- textarea for keywords
- products-per-keyword input (3 to 15)
- scrape profile editor
- grouped results page
- run status/progress
- retry failed keywords
- JSON/CSV export
- dedupe duplicate selectors
- dedupe products by URL or ASIN if possible
